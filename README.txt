Projeto Startup - Mural de Ideias para aplicações reais.

o que aprendemos:

1 - Instalar e utilizar o Angular CLI;
2 - Criar uma aplicação Angular;
3 - Estrutura da aplicação gerada pelo Angular CLI;
4 - Criar e utilizar um componente Angular;
5 - Inserimos a fonte, as cores e estilos atraves do CSS.
6 - Criar componente de formulário para adicionar novos pensamentos;
7 - Passar valores de uma propriedade do component para atributos de tag dentro do template com o uso de property binding;
8 - Mostrar valores de propriedades do component no template por meio da interpolação;
9 - Escutar eventos do template e fazer a chamada de métodos no component com o event binding;
10 - Usar a diretiva ngModel que pertence ao FormsModule para a comunicação bidirecional entre component e template;
11 - Como funcionam os diferentes tipos de comunicação entre component e template.
12 - Utilizar o RouterModule;
13 - Utilizar a diretiva <router-outlet>;
14 - Criar e configurar rotas;
15 - Navegar por meio da propriedade routerLink.
16 - Tipos diferentes de diretivas;
17 - Utilizar diretivas estruturais incluindo e excluindo elementos com ngFor e ngIf;
18 - Utilizar expressões para estilizar elementos com ngClass.
19 - Utilizar o JSON-Server simulando uma API REST;
20 - Criar uma interface para definir tipos personalizados;
21 - Entender a importância do Service e o novo decorator @Injectable();
22 - Injetar dependências.
23 - Utilizar métodos de requisição HTTP com serviço HttpClient;
24 - Utilizar o Observable no retorno dos métodos HTTP;
25 - Capturar parâmetros em rotas;
26 - Criar um CRUD.
